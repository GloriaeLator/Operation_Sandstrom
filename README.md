# Solution to the Operation_Sandstorm 

## Main Submission - Deliverables

In the /submission directory. It contains the dbg - x86_64 ELF, binary which is the debugger implemented based on the information gathered from the Vitrual Machine provided and other writeups for the problem is also present in the same directory

## Source code
The source code is present in the /machine directory. It contains the main.c - Main file. The state.h defines the state struct , and is the header of engine.c, which executes all the instructions. The io.h is the header for the io.c which has all the required functions for parsing input and loading the file. 

## Misc 
The Actual PS is present is the /other directory where modified binaries are also present. Unpatched binaries are present in the /other/examples and the challenges are present in /other/challenges directory.
